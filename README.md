# shopify-product-comparison-System
Advanced product comparison system for Shopify.

## Features

- Sticky compare bar
- Product comparison page
- Product metafield comparison
- Dynamic comparison rendering
- Persistent compare storage
- Mobile-friendly UX

## Tech Stack

- Shopify Liquid
- JavaScript
- localStorage
- Metafields

## Compared Attributes

- Pricing
- Material
- Weight
- Dimensions
- Warranty
